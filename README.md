# 🚀 Toolkit-29: Ecossistema de Engenharia de Projetos

> **O Framework definitivo para criação de estruturas de projetos blindadas, focando em Governança, Arquitetura de Software e Inteligência Artificial.**

O **Toolkit-29** é uma ferramenta de **auxílio para criação de projetos autônomos**. Ele automatiza a estruturação da raiz e das pastas do projeto, garantindo conformidade com o **Protocolo 29** e preparando seu ambiente para interagir com IAs de forma profissional.

---

## 🏗️ 1. O que compõe o Toolkit-29?

1.  **O Motor (Factory):** CLI que gera projetos baseados em templates (Next.js, Vite, etc.).
2.  **O Cérebro (29-Toolkit):** Repositório mestre de 20 Agentes Especialistas e Skills.
3.  **A Constituição (/rule):** Leis de governança que impedem códigos inseguros.
4.  **O Ingestor (29-Import):** Transpila layouts de design para código arquitetado.

---

## 🛠️ 2. Instalação (Setup em 2 Minutos)

1.  Abra o terminal na pasta do Toolkit.
2.  Rode o comando conforme seu sistema:
    *   **Mac / Linux:** `sh setup-toolkit.sh`
    *   **Windows:** `.\setup-toolkit.ps1`
3.  O instalador irá configurar os comandos globais e preparar os motores automaticamente.

---

## 🎨 3. Guia Passo a Passo: Do Design ao Código

Se você tem um arquivo `.zip` do seu layout (Stitch, etc.), o fluxo é simples:

1.  **Inicie seu Projeto:** Digite `29-init` no terminal para criar a base.
2.  **Deposite o Design:** Coloque seu arquivo `.zip` na pasta `seu-projeto/factory/inbox/design/`.
3.  **Processe:** Dentro da pasta do projeto, digite o comando:
    ```bash
    29-import
    ```
4.  **Acompanhe:** O sistema organiza o design em módulos seguros dentro de `src/modules/`.

---

## 🚀 4. Comandos Globais Disponíveis
Após a instalação, você pode usar estes comandos de **qualquer lugar**:
*   `29-init` : Cria um novo projeto blindado.
*   `29-import` : Processa e importa layouts de design (ZIP).
*   `29-toolkit` : Acessa a central mestre de agentes e inteligência.
