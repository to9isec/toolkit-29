# Modo Rápido — Quick Mode

## Objetivo

Fornecer uma visão geral rápida do documento jurídico com foco nos riscos mais relevantes.

---

## Instruções

Ao operar no modo rápido, siga estas diretrizes:

1. **Resumo executivo** — Gere um resumo de até 5 pontos com linguagem simples
2. **Tipo e partes** — Identifique o tipo de documento e as partes envolvidas
3. **Riscos principais** — Liste os 3 a 5 riscos mais relevantes encontrados
4. **Score geral** — Apresente o score de risco geral com indicador visual
5. **Recomendações** — Liste até 3 recomendações prioritárias

---

## Formato de saída

```
📋 ANÁLISE RÁPIDA — [TIPO DO DOCUMENTO]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📝 Resumo
[Resumo em até 5 pontos]

⚖️ Score de Risco: [XX]% — [NÍVEL]
[Barra visual de risco]

⚠️ Riscos Principais
1. [Risco 1]
2. [Risco 2]
3. [Risco 3]

✅ Recomendações
1. [Recomendação 1]
2. [Recomendação 2]
3. [Recomendação 3]

⚠️ Aviso Legal: Esta análise é informativa e educacional.
   Consulte um advogado para decisões jurídicas.
```

---

## O que NÃO incluir no modo rápido

- Análise detalhada de cada cláusula
- Heatmap completo
- Linha do tempo detalhada
- Comparação entre contratos
- Versão revisada das cláusulas
