# 📥 Bem-vindo ao Design Inbox!

Esta pasta é o local onde você deve colocar os seus arquivos de design para que o **Toolkit-29** possa processá-los.

## 📝 Passo a Passo:

1.  **Prepare seu Design:** Exporte seu projeto da ferramenta de design (Stitch, etc.) como um arquivo **.zip**.
2.  **Mova para cá:** Arraste e solte o arquivo `.zip` dentro desta pasta (`design/`).
3.  **Inicie o Motor:** Abra o seu terminal na pasta raiz deste projeto e digite:
    ```bash
    29-import
    ```
4.  **Aguarde a Análise:** O sistema vai abrir o ZIP, identificar a tecnologia e preparar os módulos.

---
💡 **Dica:** O Toolkit não altera o seu arquivo ZIP original. Ele cria uma cópia estruturada do código dentro da pasta `src/`, seguindo as regras de segurança e performance do seu projeto.
